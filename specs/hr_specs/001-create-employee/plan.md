## Summary

Add the Create Employee feature to the HR module so HR users can create and manage employee records with required personal, contact, identification, and versioning data. Implementation extends the Odoo `hr.employee` model to enforce uniqueness constraints (per-company user linking), create linked `resource` and `hr.version` records on creation, normalize phone numbers, and optionally create work contacts when permitted.
